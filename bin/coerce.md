# coerce

An example program to illustrate option validation and coercion.

## Usage

[options] <file ...>

## Options

* `-i, --integer <n>`: an integer argument
* `-f, --float <n>`: a float argument
* `-r, --range <a>..<b>`: a range
* `-l, --list <items>`: a list
* `-o, --optional [value]`: an optional value
